Day-1

b tag

The HTML b tag is used for specifying bold text. The b element should not be confused with the strong element (which conveys extra importance). The intention with the b tag is to markup text as bold without conveying any extra importance.

i tag

The i here stands for italic text just like b stands for bold text ( HTML b tag ). It comes under the physical style elements which are used to change the appearance of the text on a webpage.

u tag

HTML u tag is used to define a span of inline text with a non-textual annotation. It rendered as an solid underlined text, but it can be changed using CSS properties.

select tag

HTML select tag is used to create drop down list of options, which appears when the user clicks on form element, and it allows to choose one of the options.

button tag

We can also use the input tag for creating a submit button for HTML form, but then you cannot change the appearance of the button and can only change the text value which appears on the button.

hr tag

Horizontal Rule is used to drawing horizontal lines across the browser window. It is used to divide an HTML page into different sections. It does not contain any content in it, because it is an empty tag.

br tag

The br is a tag in Html. Html br tag is used to break the line of text. This tag is useful when users have to write the poem, address, and code samples.
